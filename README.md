HRMS Lite

HRMS Lite is a simple, web-based Human Resource Management System built to manage employee records and track daily attendance.
The project focuses on clean full-stack implementation, realistic functionality, and production-ready deployment.

Features

Add and view employees

Mark daily attendance (Present / Absent)

View attendance records per employee

RESTful APIs with database persistence

Tech Stack

Frontend: React (Vite), JavaScript, Axios

Backend: Node.js, Express.js

Database: MongoDB Atlas

Deployment: Vercel (Frontend), Render (Backend)

Live Links

Live App: https://hrms-lite-git-main-prakhars-projects-97ee8cfa.vercel.app/

GitHub Repo: https://github.com/prakkhar19/-hrms-lite

Run Locally (Quick)
# Backend
cd hrms-backend
npm install
npm start

# Frontend
cd hrms-frontend
npm install
npm run dev

Notes

Single admin user (no authentication)

Payroll and leave management are out of scope

Designed to be lightweight, stable, and easy to use
